Bootcamp 2021: Psychedelic Rectangles

In my project I started with a rectangle and then used loops to make a grid of rectangles to cover the whole sketch. I make smaller rectangles inside each of the rectangles in the grid. I used randon colours in fill to generate different colours for each rectangle as the sketch ran.

Check out the code on Github: https://github.com/cwatson-examples/bootcamp_project
